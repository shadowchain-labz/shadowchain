
# ShadowChain

🛡️ Privacy-first Blockchain in Rust mit:

- ✅ zk-SNARKs (Demo)
- ✅ SHDW Token System
- ✅ NFTs (mint, transfer, show)
- ✅ Smart Contract Upload (WASM)
- ✅ Token Swap (Demo)
- ✅ Terminal Explorer
- ✅ .bat CLI-Starter für Windows

## ⚙️ Start

```bash
cargo run
```

## 📦 Features testen

```bash
cargo run -- wallet
cargo run -- swap shdw 50
cargo run -- nft mint alice "Dark Angel #1"
```

## 🧪 Tests ausführen

```bash
cargo test
```
